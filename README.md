# Автоматизация окружения для Frontend разработки

## Установка

```
npm i
```

## Запуск

**Простой сервер**

```
npm run server
```

**Валидация HTML**

```
npm run validate-html
```

**Валидация JS (JSHint)**

```
npm run jshint
```

**Валидация CSS**

```
npm run validate-css
```

**Запуск DEV режима (валидаторы)**

```
npm run watch
```
